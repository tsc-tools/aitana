Aitana: access volcano data

Aitana (named after the great football midfielder Aitana Bonmati) is a project to facilitate access to different types of volcano data.

It currently only provides methods to access data from the New Zealand volcanoes Ruapehu and Whakaari/White Island.

Dependencies

  • pandas
  • requests
  • matplotlib
  • statsmodels

Installation

pip install -U aitana
